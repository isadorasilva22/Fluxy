# Fluxy

Aplicação web simples para controle financeiro pessoal, permitindo o registro de receitas e despesas, cálculo automático de saldo e visualização completa dos lançamentos.

---

## 📌 Objetivo

O projeto foi desenvolvido com o objetivo de aplicar conceitos fundamentais de desenvolvimento web (HTML, CSS e JavaScript), além de simular um sistema simples de controle financeiro.

A proposta é oferecer uma interface amigável, leve e intuitiva para organização de despesas pessoais.

---

## ✨ Funcionalidades

- ✔ Cadastro de receitas
- ✔ Cadastro de despesas
- ✔ Cálculo automático de:
  - Total de receitas
  - Total de despesas
  - Saldo disponível
- ✔ Página dedicada para visualização de todos os registros
